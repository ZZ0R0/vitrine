# Annexes — PDFs

Ce dossier contient les documents PDF associés à la vitrine.

## Fichiers prévus

| Fichier | Contenu | Statut |
|---------|---------|--------|
| `CV.pdf` | CV complet (fourni par l'humain) | ⏳ À fournir |

## Règles

- **CV.pdf** : fourni par l'humain, non généré automatiquement. À déposer ici pour qu'il soit accessible depuis la page [[à-propos/cv|CV]].
- **Pas de document confidentiel** : aucun document contenant des données client, des configs ou des informations identifiables.
- **Anonymisation** : si un PDF contient des captures, elles doivent être anonymisées selon la [[methodes/anonymisation-publication|politique d'anonymisation]].
- **Métadonnées** : vérifier et supprimer les métadonnées PDF avant commit (`exiftool -all= document.pdf`).
